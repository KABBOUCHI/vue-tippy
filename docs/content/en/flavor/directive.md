Simple and elegant, great for adding simple tooltips for any element.

Click [here](/props) to see full documentation on props.

## Basic Usage

```html
<button v-tippy="'Hello!'">
  Tippy!
</button>
```

## Advanced Usage

```html
<button v-tippy="{ content: 'Hello!', arrow: true, theme: 'light'">
  Tippy!
</button>
```
