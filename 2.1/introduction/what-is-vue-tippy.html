# What is VueTippy?

Tippy.js is a lightweight, vanilla JS tooltip library powered by Popper.js.

VueTippy is a VueJS wrapper for the Tippy.js library, which allows you to use the tooltips as a Vue component and as a directive.



It is designed to work friendly with VueJS
